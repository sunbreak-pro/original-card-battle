# 剣士カード設計ドキュメント v2

---

## システム変更点

```
■ 剣気システム
- 毎ターン開始時に剣気+1（自動獲得）
- 最大値: 10（それ以上は増えない）
- 攻撃カード: 剣気を消費する
- スキル/ガードカード: 剣気を獲得できる

■ コスト調整
- 全カードのエナジーコスト+1

■ パラメータ調整
- 全体的に50%削減

■ 剣気消費量（威力連動）
- 低威力（1-5）: 剣気-1
- 中威力（6-10）: 剣気-2
- 高威力（11+）: 剣気-3
```

---

## クラスコンセプト: 剣士

```
■ 強み
- 攻撃カードが豊富
- 瞬間火力に優れる
- バフ/デバフの組み合わせで戦略的にダメージ軽減

■ 弱み
- 防御カードが少ない
- 剣気管理が重要（攻撃に消費、防御/スキルで獲得）
```

## 剣気フロー設計

```
■ 獲得源
- 毎ターン自動: +1
- スキルカード: +2〜+5
- 一部ガードカード: 獲得なし

■ 消費先
- 攻撃カード: -1〜-3（威力連動）
- 一部スキル/ガード: -2〜-3

■ 戦略パターン
1. 序盤: スキルで剣気蓄積
2. 中盤: バフ/デバフで準備
3. 終盤: 高火力攻撃で一気に消費
```
