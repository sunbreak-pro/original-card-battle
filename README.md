### 連携のやり方

まず最初に github からリポジトリを作成(READ.me 作成にチェック)　 1.
その後 Code メニューから中央の「Add file」をクリック。 2.
アップロードかファイル作成を選択 3.
desktop に作成したリポジトリ名と同じフォルダを作成する
VSCode を開く 4.
READ.me があれば OK！ 5.
なければターミナルから [git@github.com:sunbreak-pro/roguelike-card.git] 6.
もしそれでも無理なら AI に聞く 7.
