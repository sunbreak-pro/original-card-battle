# キャンプ施設詳細設計書 (CAMP_FACILITIES_DESIGN)

## 1. 施設の全体像

キャンプは探索の準備、強化、情報の整理を行うハブエリアである。
各施設で準備を整え、ダンジョン（Battle）へと出発する。

### 施設一覧

1. **ギルド（酒場・掲示板）**: 情報収集、クエスト、イベント
2. **武具店（取引所）**: アイテム購入、不要装備の売却
3. **鍛冶屋（工房）**: 装備の強化、修理、解体
4. **神殿（聖域）**: 「魂の残滓」を使用した恒久的な強化
5. **図書館（書庫）**: デッキ編成、図鑑、戦績確認（情報の集約）
6. **倉庫（自室）**: 装備の変更（ロードアウト）

※訓練場（ステータス強化・才能管理）は現在設計から除外中。

## 2. 各施設の詳細機能

### 2.1 ギルド（酒場・掲示板）

**役割**: ゲームの進行ガイド、サブクエスト、ストーリー

- **噂話（情報収集）**: 次の探索で「特定のイベント」発生率を上げる（少額のゴールド消費）。
  - 例：「魔石の噂」→ 魔石ドロップ率 UP
  - 例：「商人の噂」→ ダンジョン内ショップ出現率 UP
- **依頼（クエスト）**: 「スライムを 10 体倒せ」等のデイリー/ウィークリー目標。達成でゴールドやアイテム入手。
- **称号受取**: `NEW_CHARACTER_SYSTEM.md` で定義された称号（剣豪、大魔導師など）の報酬受け取り。

### 2.2 武具店（取引所）

**役割**: 経済活動の循環

- **購入**: ポーション類、転移石、ランダム装備パック（コモン〜レア）の購入。
  - _重要_: `return_system.md` にある「転移石」はここで補充する。
- **売却**: ダンジョンから持ち帰った不要な装備や、余った魔石の換金。
- **日替わりセール**: ランダムで特定カテゴリが安くなる。

### 2.3 鍛冶屋（工房）

**役割**: 装備の価値向上

- **装備強化**: `EQUIPMENT_AND_ITEMS_DESIGN.md` に基づき、ゴールド＋魔石を消費して装備 Lv を上げる。
- **修理**: 耐久度（AP）が減った装備をゴールドで修理する。
- **解体**: 不要な装備を分解して、少量の「魔石」や「素材」に変換する（売却との選択肢）。

### 2.4 神殿（聖域）

**役割**: ローグライト要素（死んでも残る強さ）

- **リソース**: `return_system.md` で持ち帰った「魂の残滓」を使用。
- **魂の解放（スキルツリー）**:
  - **生存本能**: 全キャラの初期 HP +5
  - **富の加護**: 初期所持ゴールド +100
  - **鑑定眼**: 未鑑定アイテムの識別率アップ
  - **鞄の拡張**: インベントリ枠の増加
  - **運命の悪戯**: ダンジョン開始時のリロール回数増加

### 2.5 図書館（書庫）

**役割**: 知識の蓄積と戦略の構築

- **初期デッキ構成**: 次回の探索に持ち込む「初期デッキ」の確認と微調整。
  - 基本カードに加え、熟練度等でアンロックされた「ボーナスカード」を 1 枚組み込む等のカスタマイズを行う。
- **カード図鑑**: 全カードの取得状況と、現在の「熟練度」の確認。
- **魔物図鑑**: 倒した敵のステータス、ドロップ品、弱点属性の閲覧。
- **戦績記録**: 過去の探索履歴、最高到達深度、死因、スコア等の確認（倉庫から移行）。
- **歴史書**: ダンジョンで拾った手記やストーリーの閲覧。

### 2.6 倉庫（自室）

**役割**: 装備の最終調整

- **装備変更**: 持ち帰った装備（頭、胴、右、左、足、アクセ）の付け替えを行う。
  - ここで設定した装備が次回の探索時の初期装備となる。
- **ロードアウト登録**: お気に入りの装備セットを保存・呼び出しする機能。

## 3. UI/UX デザイン方針

### キャンプ画面の構成

- **背景**: 冒険者たちの拠点。
- **ヘッダー**: 所持金（ゴールド）、魔石、魂の残滓を表示。
- **メインエリア**: 横スクロールまたはマップ形式で施設アイコンを配置。
- **フッター**: 「ダンジョンへ出発」ボタン（準備完了）。

### 遷移フロー

`タイトル画面` → `キャンプ（ホーム）` → 各施設で準備 → `ダンジョン選択（出撃）`

---

## 4. データ構造とエンジニアリング要件

### データ永続化の対象

1. **インベントリ**: 所持している装備、消費アイテム。
2. **リソース**: ゴールド、魔石、魂の残滓。
3. **メタ進行度**: 神殿で解放したパッシブ効果、図鑑の解放状況、カード熟練度。
4. **プレイヤー状態**: 現在装備しているアイテム、設定された初期デッキ。

### 注意点

- **訓練場の廃止に伴う調整**:
  - 基礎ステータスの永続強化は一時的に無くし、「神殿」でのパッシブ強化に一本化する。
  - 才能（タレント）システムの実装も保留とするか、将来的に図書館か神殿に統合することを検討する。
