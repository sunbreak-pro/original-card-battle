# Complete Design: 40 Exclusive Cards for Mage

## Prerequisites

### Magic Resonance System

```
Resonance Level: 0 → 1 → 2 (Max)
Resonance increases by using the same element consecutively.
Resonance resets upon using a different element.

【Resonance Effects by Element】
Fire:
- Lv0: Normal effect
- Lv1: Normal effect / Applies Burn for 2 turns
- Lv2: Normal effect / Applies Burn for 3 turns / Applies Flame Field for 2 turns

Ice:
- Lv0: Normal effect
- Lv1: Normal effect / Applies Slow for 2 turns
- Lv2: Normal effect / Applies Slow for 3 turns / Applies Serenity for 2 turns

Lightning:
- Lv0: Normal effect
- Lv1: Normal effect / None
- Lv2: Normal effect / Applies Stun for 1 turn / Applies Thunder Realm for 3 turns

Dark:
- Lv0: Normal effect
- Lv1: Normal effect / Applies Lifesteal for 2 turns
- Lv2: Normal effect / Applies Enfeeble for 3 turns / Absorbs 40% of damage dealt as HP

Light:
- Lv0: Normal effect
- Lv1: Normal effect / Applies Purify for 1 turn
- Lv2: Normal effect / Applies Purify for 2 turns / HP Recovery +10

```
