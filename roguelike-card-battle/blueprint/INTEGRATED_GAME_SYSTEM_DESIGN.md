# 1. ゲーム全体コンセプト

## 1.1 ジャンル定義

```
ジャンル: ローグライトカードバトルRPG

【核心メカニクス】
- 深度による指数的スケーリング
- 装備の耐久度管理
- 生還判断のリスク管理
- 熟練度による長期成長
```

## 1.2 ゲームの目標

```
短期目標: 1ランで深淵を攻略して生還する
長期目標: 全カードの熟練度を上げ、才能を解放する

【勝利条件】
深淵のボスを倒して生還する

【敗北条件】
HP が 0 になり死亡する
```

## 1.3 設計哲学

```
1. リスクとリターンのトレードオフ
   - 深く潜るほど報酬は大きいがリスクも増大
   - 適切なタイミングでの撤退判断

2. リソース管理の緊張感
   - 装備の耐久度
   - カードのエナジーコスト
   - HP/防御値の管理

3. ビルド構築の自由度
   - 装備・カード・才能の組み合わせ
   - 深度適正による戦略変化

4. メタ進行の達成感
   - 熟練度によるカード強化
   - 才能解放による新しいビルド
```

# 2. コアループ

## 2.1 1ランの流れ

```
【ラン準備】
1. 装備選択（6スロット）
2. 才能選択（解放済みのもの）
3. 初期デッキ確定

【ダンジョン探索】
4. 深度1（上層）開始
   ├─ 部屋選択（戦闘/イベント/ショップ/休憩）
   ├─ カード・装備・アイテム獲得
   └─ 生還判断

5. 深度2〜4 繰り返し
   └─ 敵が強化される
   └─ 装備耐久度の管理が重要に

6. 深度5（深淵）
   └─ ラスボス戦
   └─ 生還手段なし（勝利か死亡のみ）

【ラン終了】
7. 生還
   ├─ 熟練度上昇
   ├─ 報酬獲得（魔石・魂の残滓）
   └─ 装備・耐久度を持ち越し

8. 死亡
   ├─ 装備喪失（レジェンド以外）
   ├─ 魂の残滓獲得
   └─ 熟練度はカウント
```

## 2.2 戦闘ループ

```
【ターン開始】
1. エナジー回復
2. カードドロー
3. 状態異常処理（毒ダメージなど）

【プレイヤーターン】
4. カード使用
5. 攻撃・防御・回復

【ターン終了】
6. Guard（一時盾）消滅
7. 状態異常処理（酸化ダメージなど）

【敵ターン】
8. 敵の行動
9. ダメージ処理（Guard → AP → HP）

【勝利条件】
- 敵のHP を 0 にする

【敗北条件】
- 自分のHP が 0 になる
```

# 3. 深度システム

## 3.1 深度の定義

```
深度は世界の魔力濃度を表す5段階

深度1（上層）: ×1.0  魔力
深度2（中層）: ×2.0  魔力
深度3（下層）: ×4.0  魔力
深度4（深層）: ×8.0  魔力
深度5（深淵）: ×16.0 魔力
```

## 3.2 深度の影響

```
【カードへの影響】
- 魔法カードの威力が深度倍率で増幅
- カードごとに「深度適正カーブ」あり
  └─ 浅層型: 深度1〜2で強い
  └─ 深層型: 深度4〜5で強い
  └─ 汎用型: 全深度で安定

【敵への影響】
- 深度1: 基本敵のみ
- 深度2: 重度状態異常を使用開始
- 深度3: 連携行動・自己バフ
- 深度4: アーマー貫通・高火力魔法
- 深度5: 学習AI・多回行動
```

## 3.3 深度とスケーリング

| 深度 | 名称 | 魔力倍率 | 物理/HP倍率 | 部屋数 | 生還可否 |
|---|---|---|---|---|---|
| 1 | 上層 | x1.0 | x1.0 | 6部屋 | ⭕ 可能 |
| 2 | 中層 | x2.0 | x1.2 | 7部屋 | ⭕ 可能 |
| 3 | 下層 | x4.0 | x1.5 | 8部屋 | ⭕ 可能 |
| 4 | 深層 | x8.0 | x2.0 | 9部屋 | ⭕ 可能 |
| 5 | 深淵 | x16.0 | x3.0 | 1部屋 | ❌ 不可 |

**重要:** 深淵では全ての生還手段が無効化される

# 4. ダンジョン構造

## 4.1 部屋の種類

```
【戦闘部屋】
- 通常敵（雑魚）
- エリート敵（強化版）
- ボス（階層最後）

【イベント部屋】
- ランダムイベント
- 選択肢によって結果変化
- リスク/リターンのトレードオフ

【ショップ】
- カードパック購入
- 装備購入
- アイテム購入

【休憩所】
- HP回復
- 装備修理
- カード整理

【宝箱】
- ランダム報酬
- 装備・カード・魔石

【隠し部屋】
- 特別な報酬
- 発見には条件あり
```

## 4.2 部屋数の計算

```
部屋数 = 5 + 深度レベル

深度1: 6部屋
深度2: 7部屋
深度3: 8部屋
深度4: 9部屋
深度5: 1部屋（ボスのみ）
```

## 4.3 進行ルート

```
各深度の最後にボスが配置される
ボス撃破で次の深度へ進める

【選択肢】
- 次の深度へ進む
- 生還する（深度1〜4のみ）

【深淵の特別ルール】
- 深度5に入ると生還不可能
- ボスを倒すか死ぬかの二択
```

# 6. 戦闘システム（統合版）

## 6.1 防御システムの統合

```
【3層防御構造】

1. Guard（ガード）
   - カード効果による一時的な防御壁
   - ターン終了時に消滅
   - APを守る役割

2. AP（Armor Points / アーマー）
   - 装備の耐久値
   - 戦闘後も持ち越される
   - 0になるとアーマーブレイク

3. HP（Health Points）
   - キャラクターの生命力
   - 0になると死亡
```

## 6.2 ダメージ処理の優先順位

```
敵の攻撃
   ↓
【Step 1】 Guard で受ける
   ↓（Guardがなくなったら）
【Step 2】 AP で受ける
   ↓（APがなくなったら）
【Step 3】 HP で受ける

【特殊ルール: アーマーブレイク】
AP = 0 の状態では
→ ダメージの50%がGuardを無視してHPへ直撃
```

## 6.3 状態異常システム

```
【重症度進化】
軽度(Light) → 重度(Heavy)

同じ状態異常を重ねがけすると進化
重度でさらに重ねても持続ターンのみ延長

【主要な状態異常】
- 毒/猛毒: ターン開始時にHP減少
- 出血/大出血: 行動時にHP減少
- 酸化/溶解: AP減少 + Guard効果半減
- よろめき/気絶: Energy減少 / 行動不能
- 凍結/氷獄: 手札制限
- 弱体/脆弱: 被ダメージ増加
- 脱力/無力: 与ダメージ減少
```

## 6.4 ターン進行タイミング

```
【Turn Start Phase】
- エナジー回復
- カードドロー
- 毒/猛毒ダメージ

【Player Action Phase】
- カード使用
- 出血/大出血ダメージ

【Turn End Phase】
- 手札を捨てる
- Guard消滅（0になる）
- 酸化/溶解ダメージ

【Enemy Action Phase】
- 敵の行動
- バフ/デバフ減少
```

# 7. 装備システム

## 7.1 装備スロット（6箇所）

```
1. 頭（Head）
2. 胴（Body）
3. 右手（Right Hand）
4. 左手（Left Hand）
5. 靴（Boots）
6. アクセサリ（Accessory）
```

## 7.2 装備の基本パラメータ

```
【ステータス補正】
- ATK補正: 与ダメージ増加
- DEF補正: 被ダメージ軽減
- エナジー補正: 初期エナジー増加

【装備固有値】
- AP（最大耐久値）
- 現在耐久値（戦闘で減少）
- レアリティ（コモン〜レジェンド）
```

## 7.3 装備の特性（Trait）

```
【パッシブ効果】
- カードドロー+1
- 初手確定（特定カード）
- 深度補正+1
- クリティカル率上昇
- 状態異常耐性
- ターン開始時シールド付与

【レジェンド装備の特別ルール】
- 死亡時も失われない
- 高い基礎ステータス
- 強力な固有特性
```

## 7.4 装備の耐久度管理

```
【減少タイミング】
- APが削られると耐久度減少
- アーマーブレイク後は急速に減少

【修理方法】
- 休憩所で修理
- 修理カードの使用
- 魔石を使った修理

【戦略的意義】
- 耐久度が低いと生還判断に影響
- 高耐久装備の価値
- 修理タイミングの重要性
```

# 8. カードシステム

## 8.1 カードの4カテゴリ

```
1. 物理攻撃（Physical Attack）
   - 威力 = 基本値 × (1 + ATK補正)
   - 深度の影響: 小
   - 特徴: 安定したダメージ

2. 魔法攻撃（Magic Attack）
   - 威力 = 基本値 × 魔力倍率 × (1 + 魔法補正)
   - 深度の影響: 大
   - 特徴: 深層で爆発的強化

3. 守り（Defense）
   - Guard付与
   - ダメージ軽減
   - AP修復

4. 回復（Heal）
   - HP回復
   - 状態異常解除
   - バフ付与
```

## 8.2 カードの共通プロパティ

```
【基本情報】
- 名称
- コスト（エナジー）
- カテゴリ
- レアリティ

【効果】
- 威力/効果値
- 対象（単体/全体/自分）
- 追加効果（状態異常など）

【深度適正】
- 深度適正カーブ
- 最適深度での強化率

【強化】
- 魔石による段階強化
- 超魔石による最終強化
```

## 8.3 深度適正カーブ

```
【浅層型カード】
深度1: 130%
深度2: 110%
深度3: 90%
深度4: 70%
深度5: 50%

【汎用型カード】
全深度: 100%

【深層型カード】
深度1: 50%
深度2: 70%
深度3: 90%
深度4: 120%
深度5: 150%

【狂気型カード】
深度1〜4: 80%
深度5: 200%（デメリット付き）
```

# 9. 生還システム

## 9.1 生還方法の2種類

```
1. 帰還ルート（徒歩帰還）
   - 来た道を戻る
   - 道中で敵と遭遇（確率的）
   - 報酬: 100%（減少なし）

2. 転移石（即時帰還）
   - アイテムを消費して即座に帰還
   - 3種類の転移石あり
   - 報酬: 減少あり
```

## 9.2 転移石の種類

| 種類 | 報酬倍率 | 戦闘中使用 | レアリティ |
|---|---|---|---|
| 通常転移石 | 70% | ❌ 不可 | コモン |
| 祝福転移石 | 80% | ❌ 不可 | レア |
| 緊急転移石 | 60% | ⭕ 可能 | エピック |

## 9.3 帰還ルートの遭遇率

```
【動的遭遇率システム】
進行するほど遭遇率が減少（拠点に近づくほど安全）

深度1: 70% → 20%
深度2: 75% → 25%
深度3: 80% → 30%
深度4: 85% → 35%

計算式:
現在遭遇率 = 初期遭遇率 - (進行度% × 0.5)
```

## 9.4 深淵の特別ルール

```
【深度5（深淵）の制約】
❌ 帰還ルート使用不可
❌ 全ての転移石が無効化
❌ 途中撤退不可能

→ ボスを倒すか死ぬかの二択

【突入前の警告】
深淵へ入る前に警告が表示される
プレイヤーは覚悟を決めてから突入
```

## 9.5 生還と死亡の比較

| 項目 | 生還 | 死亡 |
|---|---|---|
| 装備 | 全て保持 | 喪失（レジェンド以外） |
| 装備耐久度 | そのまま | - |
| 熟練度 | カウント記録 | カウント記録 |
| 魔石 | 方法により減少 | 0 |
| 魂の残滓 | 方法により減少 | 通常獲得 |
| 装備の痕跡 | なし | 獲得 |

# 10. 経済システム

## 10.1 主要リソース

```
【魔石（Magic Stones）】
- サイズ: 極小/小/中/大/極大
- 用途: カード強化、装備強化
- 入手: 戦闘、宝箱、ボス報酬

【魂の残滓（Soul Remnants）】
- 用途: 装備の痕跡との交換
- 入手: 生還時（減少あり）、死亡時

【装備の痕跡（Equipment Trace）】
- 用途: 過去に失った装備の再取得
- 入手: 死亡時に装備が変換される
```

## 10.2 ショップシステム

```
【パック販売制】
カードは単品では売らず、パックで販売

- 浅層パック: 深度1〜2向けカード
- 中層パック: 深度2〜3向けカード
- 深層パック: 深度3〜4向けカード
- 深淵パック: 深度5向けの強力な1枚
- 装備パック: ランダム装備
- 熟練度パック: 熟練度のみ獲得

【価格】
深度が深いほどパック価格が上昇
熟練度が高いほどカードの出現率上昇
```

## 10.3 魔石の使用優先度

```
【推奨優先順位】

1. コアカードの強化
   → デッキの軸となるカードを優先

2. 装備の修理
   → 耐久度が20%以下なら緊急修理

3. 装備の強化
   → AP最大値を上げる

4. サブカードの強化
   → 余裕があれば
```

# 11. メタ進行（熟練度・才能）

## 11.1 熟練度システム

```
【基本概念】
カードを所持・使用するほど熟練度が上昇
生還時のみ熟練度が記録される

【熟練度レベル】
Lv1: 初回獲得
Lv2: 使用回数 10回
Lv3: 使用回数 30回
Lv4: 使用回数 60回
Lv5: 使用回数 100回（才能解放）
```

## 11.2 熟練度の恩恵

```
【Lv2の恩恵】
- ショップ出現率 +20%
- 購入コスト -10%

【Lv3の恩恵】
- ショップ出現率 +40%
- 購入コスト -20%
- 深度適正 +5%

【Lv4の恩恵】
- ショップ出現率 +60%
- 購入コスト -30%
- 深度適正 +10%
- 魔石強化費 -20%

【Lv5の恩恵（才能解放）】
- カードを所持していなくても効果発動
- パッシブ効果として常駐
```

## 11.3 才能（Talent）システム

```
【才能の種類】

攻撃系:
- 物理攻撃初回0コスト
- 魔法カード威力+20%
- クリティカル率+15%

防御系:
- ターン開始時Guard+4
- 被ダメージ-15%
- 状態異常耐性+50%

リソース系:
- 初期手札+1
- 毎ターンエナジー+1
- カードドロー+1

特殊系:
- 深度を+1段階扱い
- 回復カードがGuard付与
- 装備耐久度減少-30%
```

## 11.4 才能ビルド例

```
【物理特化ビルド】
才能1: 物理攻撃初回0コスト
才能2: クリティカル率+15%
才能3: 初期手札+1
→ 序盤から圧倒的な手数

【魔法深層ビルド】
才能1: 魔法カード威力+20%
才能2: 深度を+1段階扱い
才能3: 毎ターンエナジー+1
→ 深層・深淵で爆発的火力

【防御耐久ビルド】
才能1: ターン開始時Guard+4
才能2: 装備耐久度減少-30%
才能3: 被ダメージ-15%
→ 長期戦に強い安定型
```

# 12. システム間の相互作用

## 12.1 装備 × 生還

```
【装備耐久度が生還判断に影響】

耐久度 > 40%:
→ 帰還ルート推奨
→ 道中戦闘を耐えられる

耐久度 20〜40%:
→ 転移石推奨
→ 装備保護が優先

耐久度 < 20%:
→ 緊急転移石推奨
→ 次戦闘で破損の危険
```

## 12.2 深度 × カード

```
【深度適正による戦略変化】

浅層（深度1〜2）:
→ 浅層型カード・物理攻撃が強い
→ 安定した戦闘

深層（深度3〜4）:
→ 深層型カード・魔法攻撃が強い
→ カードの選択が重要

深淵（深度5）:
→ 狂気型カードのリスク/リターン
→ 最適化されたデッキが必要
```

## 12.3 熟練度 × 経済

```
【熟練度が高いほど経済的に有利】

出現率上昇:
→ 欲しいカードが出やすい
→ 効率的なデッキ構築

コスト削減:
→ 魔石の節約
→ より多くのカード強化

才能解放:
→ カード購入不要
→ リソースを他に使える
```

## 12.4 状態異常 × 防御

```
【溶解状態とGuard効率】

通常時:
Guard 10 → 10ダメージ防ぐ

溶解状態:
Guard 10 → 5ダメージしか防げない
→ APへのダメージが増加

【戦略的対応】
溶解中は防御カードの効率が悪い
→ 回復カードで解除
→ 攻撃に集中して早期決着
```

# 14. 矛盾の解消と設計改善

## 14.1 主要な矛盾点の修正

```
【矛盾1: 防御システムの統合】
問題: ゲームコンセプトの「シールド」とバトルロジックの「Guard/AP」が混在
解決: 以下の3層構造に統一
  - Guard（一時盾、ターン終了で消滅）
  - AP（装備耐久、持ち越し可能）
  - HP（生命力）

【矛盾2: 気絶システムの整理】
問題: ゲームコンセプトとバトルロジックで気絶条件が異なる
解決: 状態異常としての気絶に統一
  - よろめき（軽度）: Energy -1
  - 気絶（重度）: 行動不能

【矛盾3: 生還システムの報酬】
問題: 報酬倍率やペナルティが不明確
解決: 明確な数値定義
  - 帰還ルート: 100%
  - 通常転移石: 70%
  - 祝福転移石: 80%
  - 緊急転移石: 60%

【矛盾4: 深淵の位置づけ】
問題: 深淵での生還可否が曖昧
解決: 深淵では全ての生還手段が無効と明確化
```

## 14.2 バランスの改善

```
【改善1: 装備耐久度の重要性向上】
- APがダメージ吸収の主力
- アーマーブレイクのペナルティを明確化
- 修理タイミングの戦略性向上

【改善2: 深度適正の明確化】
- カードごとに最適深度を明示
- 深度適正カーブを数値化
- プレイヤーの理解を助ける

【改善3: 熟練度の価値向上】
- 才能システムとの連携
- 長期目標の明確化
- リプレイ性の向上

【改善4: 生還判断の面白さ】
- 動的遭遇率システム
- 装備耐久度との連携
- リスク/リターンの可視化
```

## 14.3 システム間の整合性確保

```
【確保した整合性】

1. 時間軸の統一
   - ラン内（短期）
   - 装備成長（中期）
   - 熟練度成長（長期）

2. リソースフローの統一
   - 獲得 → 強化 → 消費
   - 循環構造の確立

3. リスク/リターンの統一
   - 深度が深いほど高リスク/高リターン
   - 生還方法によるトレードオフ
   - 装備耐久度管理の緊張感

4. 成長曲線の統一
   - 初心者でも楽しめる
   - 上級者には深い戦略
   - 長期プレイでも飽きない
```

# 16. まとめ

## 16.1 設計の核心

```
このゲームの核心は3つの緊張感

1. 戦闘の緊張感
   - 3層防御の管理
   - 装備耐久度の減少
   - 状態異常の脅威

2. 探索の緊張感
   - 深く潜るべきか撤退すべきか
   - 装備修理のタイミング
   - リソース配分の判断

3. 成長の達成感
   - 熟練度の蓄積
   - 才能の解放
   - ビルドの完成
```
