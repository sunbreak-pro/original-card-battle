## 1.4 シールドシステム

### シールドの効果
```
シールドあり: HPダメージ半減
シールド破壊: 残りダメージ等倍 + 気絶判定
```

### シールドの持続性（重要）
```
【ターン終了時】
シールド持ち越し → 消えない

【戦闘終了後】
シールド持ち越し → 消えない

【理由】
シールドは装備の魔力効果
装備が健在な限り維持される
```

### シールドの減少
```
1. 敵の攻撃
2. カード効果での消費
3. 装備破損（新規付与停止）
```

### シールド値（1.5倍強化済み）
```
【コスト1】
鉄壁: 9
魔力障壁: 12
緊急シールド: 15

【コスト2】
大盾: 23
完全防御: 30
永続結界: 12 + 毎ターン+6

【コスト3】
不落の城壁: 75
```

## 1.5 気絶システム

### 気絶の発生条件
```
シールド破壊攻撃を受けた時
→ 気絶確率 = 100% - 気絶耐性%
```

### 気絶の効果
```
効果: 次の自ターン行動不可
期間: 基本1ターン（ボス・エリートは2〜4ターン）
ダメージ: 通常通り（追加ペナルティなし）
シールド: 残る（次ターンに有効）
```

### 気絶耐性の獲得
```
【装備】
頭: +5〜50%
胴: +5〜60%
左手(盾): +10〜50%
靴: +5〜60%

【アクセサリ】
不屈の護符: +30〜60%
鋼鉄の心臓: 気絶無効
覚醒の宝珠: 気絶無効

【カード】
不動の構え: 気絶無効(このターン)
鉄壁の誓い: 気絶耐性+50%(3ターン)
不屈の魂: 気絶解除 + 気絶無効(3ターン)

【アイテム】
不屈の薬: 気絶無効(1戦闘)
```

### 気絶耐性の累積
```
加算方式
100%以上で完全無効

例:
頭+30% / 胴+40% / 盾+25% / アイテム+70%
= 165% → 気絶無効
```

## 2.3 熟練度システム（重要）

### 熟練度の計算

```
【カウント方法】
使用回数のみでカウント
1回使用 = 熟練度+1

【同種カードの扱い】
同じカード種類は使用回数・熟練度を共有
例: 斬撃×3枚 → どれを使っても「斬撃」+1

【成長タイミング】
ラン中リアルタイム成長
→ 戦闘中でも強化
→ 即座に反映
```

### 強化段階

```
Lv0(初期): 0〜7回使用
Lv1(熟練): 8〜15回使用 → 効果+20%
Lv2(達人): 16〜23回使用 → 効果+40%
Lv3(才能): 24回使用 → 才能カード化
```

### 才能化

```
【条件】
24回使用達成

【処理】
1. 元カード全て消失
2. 才能カード×1に変換

4. 優先ドロー（戦闘開始時に手札確定）

【特徴】
コスト: 0
効果: 強力
使用制限: 1戦闘1回（一部を除く）
除去: 不可
```

### 才能カードの例

## 4.2 ダメージ計算

### 物理ダメージ

```
威力 × 深度適正% × (1 + ATK補正) × (1 + 装備補正) × 熟練度倍率

例:
斬撃(威力8 / 浅層型130% / Lv1+20%)
ATK+30% / 装備+20%
→ 8 × 1.3 × 1.3 × 1.2 × 1.2 = 19.4 → 19
```

### 魔法ダメージ

```
基本威力 × 魔力倍率 × 深度適正% × (1 + 魔法補正) × (1 + 装備補正) × 熟練度倍率

例:
火球(基本10 / 深層型140% / Lv1+20%)
深淵(×16) / 魔法補正+30% / 装備+40%
→ 10 × 16 × 1.4 × 1.3 × 1.4 × 1.2 = 488
```

### シールド計算

```
基本値 × (1 + DEF補正) × (1 + シールド効果補正) × 熟練度倍率

例:
大盾(基本23 / Lv1+20%)
DEF+30% / シールド効果+60%
→ 23 × 1.3 × 1.6 × 1.2 = 57
```
