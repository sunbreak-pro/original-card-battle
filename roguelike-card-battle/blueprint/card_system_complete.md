# カードシステム全体設計

## 【必須】設計方針

### **カード総数と分布**

```
合計: 90枚

カテゴリ別:
- 物理攻撃: 30枚
- 魔法攻撃: 25枚
- 守り: 20枚
- ヒール: 15枚
```

### **コスト分布の基本原則**

```
コスト0: 5% (特殊カード / 才能カード専用)
コスト1: 40% (軽量 / テンポ重視)
コスト2: 35% (標準 / バランス型)
コスト3: 15% (重量 / 高火力)
コスト4-5: 5% (超重量 / フィニッシャー)

理由:
- コスト1が多い → 序盤の安定性
- コスト2が標準 → 中盤の柔軟性
- コスト3以上は少数 → 特別な選択肢
```

### **深度適正カーブの分布**

```
浅層型: 25% (序盤安定)
中立型: 35% (汎用性)
深層型: 30% (後半爆発)
狂気型: 5% (ハイリスク)
逆境型: 5% (特殊戦術)
```

### **レアリティ区分**

```
コモン: 60% (基本カード)
レア: 25% (強化カード)
エピック: 12% (特殊効果)
レジェンド: 3% (最強カード)

レアリティとコストは独立
→ コスト1でもレジェンドあり
→ コスト4でもコモンあり
```

### **魔石強化の原則**

```
通常 → 魔石強化(+1段階):
- 威力+50%
- 効果追加(デバフ / 範囲 / 特殊)

魔石強化 → 超魔石(+2段階):
- 威力+100%(通常比)
- 効果大幅強化
- 新能力付与
```

## 【推奨】シナジー設計

### **物理コンボ**

```
【連撃特化】
連撃 + 練気斬 + 猛攻 + 剣神の構え(才能)
→ 低コスト連続攻撃で剣気蓄積

【貫通特化】
突き + 貫通撃 + 鎧砕き + 貫通の心得(才能)
→ シールド無視で安定ダメージ

【自傷火力】
捨て身 + 渾身の一撃 + 光の加護
→ HPを削って超火力、回復で調整
```

### **魔法コンボ**

```
【火傷特化】
火球 + 火炎放射 + 業火 + 炎の壁
→ 火傷スタックで持続ダメージ

【凍結制御】
氷結 + 吹雪 + 絶対零度
→ 敵の行動を完全停止

【連鎖魔法】
連鎖魔法 + 火球 + 魔力爆発
→ 威力倍増コンボ
```

### **守り+回復コンボ**

```
【持続防御】
再生シールド + 永続結界 + 生命の奔流
→ 毎ターン自動回復で長期戦

【反撃型】
反撃の盾 + 反射障壁 + 鏡の盾
→ 攻撃されるほど強い

【完全防御】
不落の城壁 + 完全復活 + 聖域
→ 高コストだが鉄壁
```
